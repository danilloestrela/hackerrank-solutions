# Lonely Integer

## Problem Description

Given an array of integers, where all elements but one occur twice, find the unique element.

**Example:** 
`arr = [1,2,3,4,3,2,1]`

- The unique element is `4`

Complete the` lonelyinteger` function in the editor below.
